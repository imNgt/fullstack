# Day 21：总复盘

### 📚 21 天学习总结

#### 1. 知识体系梳理

**Java 核心知识点：**

| 模块 | 核心内容 |
|-----|---------|
| **基础语法** | 类、对象、继承、封装、多态 |
| **泛型** | 类型安全、通配符、类型擦除 |
| **Lambda & Stream** | 函数式编程、流式操作 |
| **集合框架** | HashMap、ArrayList、LinkedList |
| **并发编程** | 线程、线程池、线程安全 |
| **反射** | 运行时获取类信息、动态操作对象 |
| **注解** | 元数据、自定义注解、注解处理器 |
| **设计模式** | 单例、工厂、策略、代理 |

**Spring Boot 核心知识点：**

| 模块 | 核心内容 |
|-----|---------|
| **IoC/DI** | 依赖注入、Bean 生命周期 |
| **MVC** | RESTful API、参数绑定、统一返回 |
| **AOP** | 切面编程、自定义注解 |
| **事务管理** | @Transactional、传播行为、隔离级别 |
| **数据访问** | JPA、MyBatis Plus、连接池 |
| **安全** | Spring Security、JWT 认证 |
| **缓存** | Spring Cache、Redis 集成 |
| **异步** | @Async、@Scheduled、消息队列 |

**项目实战核心技能：**

| 技能 | 说明 |
|-----|------|
| **项目架构** | 三层架构、DTO 设计、统一返回 |
| **权限管理** | RBAC、细粒度权限控制 |
| **性能优化** | 索引、缓存、批量操作 |
| **监控告警** | Actuator、自定义指标 |
| **部署上线** | Docker、Docker Compose、Nginx |
